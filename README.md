# zen

## 简介

zen 是一款 3D 图表库, 旨在通过为传统 2D 图表多增加一个维度的方式来更为高效的展示更多数据, 并且更便于进行数据间的比较

目前支持以 React 组件的形式添加入已有的项目中

[文档地址](https://zen.yuleiz.com/) (推荐使用电脑查看, 目前文档和例子页面还未来得及做手机适配, 电脑不在身边的话也可以手机扫码大致看一下我们的主页~)

[备用文档地址](https://zen2.netlify.app/)
